LTSP Thin Client Server

College Project | 2024

I set up a Linux Terminal Server Project (LTSP) using Ubuntu 18, turning a single server into a hub for multiple thin clients, perfect for resource-efficient user management.

Approach

  • Installed and configured Ubuntu 18 as the server OS.
  • Set up LTSP to distribute resources to lightweight client devices.
  • Managed user accounts and permissions for secure access.
  • Tested performance with multiple simultaneous connections.

Tools Used

Ubuntu 18, LTSP, Linux Terminal Commands

Outcome

Created a cost-effective, scalable server solution that could support a classroom or small office with minimal hardware.
